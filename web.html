Video Game Quiz and Drag & Drop Quiz

I created a "Video Game Pop Quiz" using Notepad++ for the .js and .html files, hosting it on GitHub. To enhance the visual appeal of the quiz, I added a background image. I updated the quiz title to “Video Game Pop Quiz” and changed the font to Arial for improved readability, using the code font-family: 'Arial', cursive;. In the JavaScript section, I customized all questions and answers to align with the video game theme and increased the total number of questions to 10. Additionally, I added a range of user engagement compliments, such as "Amazing!", "Nailed it! You're on fire!".

The assignment involved creating a Video Game Drag & Drop Quiz using Brackets for HTML and CSS modifications, and GitHub for hosting. In terms of CSS changes, the font was updated to 'Times New Roman' to enhance readability and provide a classic look. The background color was changed to a warm shade (#b3b69a), which helps the content stand out without being overwhelming. Additionally, the drag container's background was updated to #317082, making the draggable items more prominent and engaging for users. On the HTML side, the character boxes were updated to feature popular appealing to a broader audience. The box identifiers were modified to match their respective games', which improved navigation clarity. The instructional text was also rewritten to better align with the new character selections and gameplay, ensuring that users received straightforward guidance. Lastly, the drag-in boxes were updated to display the correct characters in their respective game locations, enhancing both functionality and user interaction.
